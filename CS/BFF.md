## BFF (Backend for Frontend)

- 프론트엔드 친화적인 API 서버
- 예: boolean 값들로 이뤄진 raw한 데이터를 BFF에서 프론트 친화적인 데이터로 가공하여 프론트엔드에서는 UI 표현하는 데 집중하도록 한다.
